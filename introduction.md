# Vode (View Your Code) 产品白皮书与功能概览

## 1. 概述：重塑代码的认知方式

**Vode** (View Your Code) 是一款综合性的代码可视化与分析平台。作为开发者，在面对庞大且复杂的 Python 程序（特别是基于 PyTorch 构建的深度学习模型）时，仅凭阅读纯文本代码往往难以构建清晰的全局执行逻辑。

Vode 旨在解决这一痛点，它提供了一套完整的静态与动态捕获机制，并结合静态图像导出与交互式可视化界面，帮助您直观地“看透”代码的物理执行结构。更进一步，Vode 正在规划一种全新的范式：在未来，您将能够通过可视化拖拽的方式“创造”模型与代码。

---

## 2. 核心交互逻辑：Input → Op → Output 模型

在使用 Vode 时，您无需学习复杂的图论概念。系统将任何 Python 程序优雅地抽象为统一的 **输入 → 操作 → 输出 (Input → Op → Output)** 范式：

* **输入 (Input)**：流入节点的函数参数、张量 (Tensor) 数据或任何前置输入值。
* **操作 (Op)**：程序中的执行实体，无论是顶层的业务函数、中间层的网络模块 (Module)，还是底层的数学算子 (Operator)。
* **输出 (Output)**：操作执行完毕后的返回值、张量输出或最终结果。

**使用者体验**：这种范式具有**递归可展开性**。在交互界面中，当您看到一个宏观的 `ResNet` 节点时，只需轻轻点击，即可将其展开为内部的 `Bottleneck` 模块；再次点击，即可继续深入至 `Conv2d` 与 `ReLU` 算子。您可以自由决定观察代码的宏观架构或微观细节。

## 3. 分析现有代码：Vode 的“看”能力

Vode 为您提供了极具弹性的分析工具箱。日常工作流通常从执行捕获命令开始，随后选择合适的渲染方式。工具提供了 4 种核心的“捕获×展示”组合。

### 3.1 捕获阶段 (Capture Phase)

您可以通过简单的命令行指令（如 `vode trace script.py`）启动捕获引擎。Vode 会在后台拦截程序运行状态，并生成一份标准化的 `.vode.json` 追踪文件。您可以选择以下两种模式：

* **函数流捕获 (Function Flow)**：
* **适用场景**：当您需要排查 Python 侧的业务逻辑、函数调用链或性能瓶颈时。
* **捕获内容**：从 `main()` 入口到叶子函数（例如 `torch.nn.Module` 的具体方法）的完整调用栈与执行时间。系统会自动过滤掉过于底层的 C++ 算子以避免信息过载。

* **计算流捕获 (Computation Flow)**：
* **适用场景**：当您只关注深度学习模型本身的数学架构，希望排除繁杂的 Python 控制流代码时。
* **捕获内容**：纯粹的计算图拓扑，以网络模块（如 `Linear`, `Conv2d`）为原子节点，揭示数据在模块间的真实流向。

**深度洞察（可选数据捕获）**：
通过追加 `--capture-data` 参数，Vode 能够进一步抓取运行时的数据细节。在函数流中，您可以查看具体的传入参数与返回值；在计算流中，您可以洞察张量的形状 (Shapes)、数据类型 (Dtypes)、所在设备 (Devices) 以及统计学特征（最小值、最大值、均值等）。

### 3.2 渲染与分析阶段 (Display Phase)

获得追踪文件后，您可以通过以下两种方式对结构进行分析：

* **交互式视图 (Interactive View - `vode view`)**：
* **界面体验**：在浏览器或 VSCode 的 Webview 面板中打开一个基于节点编辑器的交互式画布。
* **全局导航**：支持在大规模计算图中进行平移、缩放，并提供精准的名称搜索与节点类型过滤功能。
* **详情侧边栏**：点击任意节点，侧边栏将为您详细展示该节点的源码物理路径、代码行号、执行耗时、参数数量、显存占用以及张量的详细数据特征。
* **视觉编码**：不同的节点类型将以不同的颜色区分；数据流转的边 (Edge) 粗细将与数据量大小成正比；节点大小可映射为执行耗时，帮助您一眼识别关键路径。

* **静态图像导出 (Static Export - `vode export`)**：
* **使用场景**：当您需要将模型架构或调用栈嵌入技术文档或学术论文时。
* **功能**：允许您指定最大展示层级 (Depth)，将庞大的网络提炼为结构清晰的出版级图片 (PNG/SVG 格式)。

## 4. 展望未来：从“分析”到“创造”

Vode 的产品蓝图不仅止步于逆向工程。在未来的版本迭代中，Vode 将引入一系列面向正向工程与深度性能调优的高阶特性。

### 4.1 可视化编程环境 (Visual Programming)

这是 Vode 最具突破性的未来特性，旨在提供类似 Scratch 的沉浸式模型构建体验：

* **工作台布局**：界面左侧提供丰富的预置模块库（涵盖各类型的神经网络层与算子），右侧为广阔的构建画布。
* **无代码构建**：作为使用者，您可以直接将模块拖拽至画布，通过鼠标连线即可定义复杂的神经网络拓扑结构。
* **参数配置与代码生成**：在属性面板精调模块参数后，Vode 能够将您构建的可视化图形，一键翻译并生成为标准、可执行的 Python 代码。

### 4.2 深度 VSCode 原生集成 (VSCode Extension)

为了提供无缝的开发者体验，Vode 将推出专属的 VSCode 插件：

* 您可以通过命令面板 (Command Palette) 直接唤起捕获与追踪任务。
* 在交互面板中点击任意可视化节点，编辑器将立即跳转至对应的源代码行。
* 支持在代码编辑器内部提供内联注释 (Inline annotations)，直接在源码旁显示函数的执行耗时。
